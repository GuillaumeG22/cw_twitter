What does it do?¶

With Twitter Feed you can show tweets from a specific user (timeline) or query on the frontend. This extension works with the new 1.1 version of the Twitter API so you need to make a Twitter App on dev.twitter.com.

Bugs?¶

If you find any bugs, please email me or, even better, fork the Mercurial repository on BitBucket, fix it yourself and make a pull request :).
